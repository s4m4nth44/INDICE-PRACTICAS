# Práctica 13 - Motor DC

## Actividad 1: Control de velocidad de un motor DC

Implementación de un sistema para controlar la velocidad de giro de un motor de corriente directa utilizando un circuito driver basado en transistores BJT.

---

## Actividad 2: Control de sentido de giro y velocidad con puente H

Implementación de un sistema para controlar el sentido de giro y la velocidad de un motor de corriente directa mediante un puente H, permitiendo invertir la dirección del motor y regular su velocidad.
