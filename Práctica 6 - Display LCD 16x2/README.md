# Práctica 6 - Display LCD 16x2

## Clase: Hello World

Implementación de un programa para mostrar el mensaje **"Hello World!"** en un display LCD 16x2 utilizando el microcontrolador PIC16F887.

---

## Actividad 1: Carácter personalizado y animación

Creación de un carácter especial personalizado en el display LCD y desarrollo de una animación que se ejecuta al presionar un botón.
