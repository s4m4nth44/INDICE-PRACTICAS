# Práctica 8 - Múltiple Lectura Analógica

## Clase: Lectura de dos potenciómetros

Implementación de un programa para leer el valor analógico de dos potenciómetros mediante el convertidor ADC del PIC16F887 y mostrar sus respectivos voltajes en un display LCD 16x2.

---

## Actividad 1: Visualización de múltiples lecturas analógicas

Implementación de un sistema que permite alternar, mediante botones, la visualización del valor de cada potenciómetro en formato ADC, Voltaje (V) y Porcentaje (%), así como seleccionar cuál de los dos potenciómetros mostrar en el display LCD.
