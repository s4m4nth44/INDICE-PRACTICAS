## Actividad 1: Contador de 0 a 99

Implementación de un contador decimal de 0 a 99 utilizando dos displays de 7 segmentos, con botones para incrementar, decrementar y reiniciar el conteo.

En esta carpeta se mostrará la evidencia de la actividad realizada, incluyendo esquemático, código utilizado y fotografía/video de su funcionamiento

Materiales:

- PIC16F887
- Protoboard
- 2 Displays de 7 segmentos
- Resistencias de 220 Ω (una por segmento)
- 3 Botones pulsadores
- Cristal
- Fuente de 5 V
- Cables
