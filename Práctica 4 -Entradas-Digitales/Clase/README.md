## Actividad en Clase: Control de LEDs con botones
Usar 3 botones conectados al puerto B para encender 3 LEDs conectados en otro puerto. Utilizar las resistencias de Pull-up internas.

En esta carpeta se mostrará la evidencia de la actividad realizada, incluyendo esquemático, código utilizado y fotografía/video de su funcionamiento

---

Materiales:

- PIC16F887
- Protoboard
- 3 LEDs
- 3 Resistencias de 220 Ω
- 3 Botones pulsadores
- Cristal de 16 MHz 
- Fuente de alimentación de 5 V
- Cables de conexión
