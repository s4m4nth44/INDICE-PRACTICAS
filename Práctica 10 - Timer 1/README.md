# Práctica 10 - Timer 1

## Clase: Tiempo de ejecución con Timer1

Implementación de un programa para medir y mostrar el tiempo de ejecución del microcontrolador utilizando el módulo Timer1 del PIC16F887 en un display LCD 16x2.

---

## Actividad 1: Lectura de voltaje y tiempo de ejecución

Implementación de un sistema que permite mostrar simultáneamente el voltaje de un potenciómetro y el tiempo de ejecución del microcontrolador utilizando el Timer1 en un display LCD 16x2.
