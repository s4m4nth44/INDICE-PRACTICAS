# Práctica 14 - Servomotores

## Actividad 1: Control de giro de un servomotor

Implementación de un sistema para controlar el ciclo de giro de un servomotor, realizando un recorrido automático desde 0° hasta 180° y de regreso mediante el uso de señales PWM.

---

## Actividad 2: Control de ángulo mediante potenciómetro

Implementación de un sistema que utiliza un potenciómetro para controlar el ángulo de giro de un servomotor, ajustando su posición entre 0° y 180° de forma proporcional.
