# Práctica 11 - PWM por Hardware y PWM por Software

## Actividad 1: Control de intensidad de un LED mediante PWM por hardware

Implementación de un sistema que utiliza un potenciómetro para controlar la intensidad luminosa de un LED mediante el módulo PWM por hardware del PIC16F887.

---

## Actividad 2: Control de intensidad de dos LEDs

Implementación de un sistema que utiliza dos potenciómetros para controlar de manera independiente la intensidad de dos LEDs, empleando PWM por hardware para uno de ellos y PWM por software para el otro.
