# Práctica 7 - Lectura Analógica

## Clase: Lectura de voltaje con potenciómetro

Implementación de un programa para leer el valor analógico de un potenciómetro mediante el convertidor ADC del PIC16F887 y mostrar el voltaje correspondiente en un display LCD 16x2.

---

## Actividad 1: Visualización de ADC, Voltaje y Porcentaje

Implementación de un sistema que permite alternar, mediante un botón, la visualización del valor del potenciómetro en formato ADC, Voltaje (V) y Porcentaje (%) en el display LCD.
